→Projects

Some of my projects are available on my github page. They include:

  • Demo Selenium Framework - link to github - Scalable Selenium framework for an e-commerce website. Built with Maven and TestNG using Allure reporting. Written in Java.
  • Demo Rest-assured Framework - link to github - Scalable and extensible Rest-assured framework that tests a Spotify playlist API. Built with Maven and TestNG using Allure reporting. Written in Java.
  • Demo Cypress Framework - link to github - Demo Cypress framework for a medium.com clone website. GUI and API testing. Written in Javascript.
